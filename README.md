# Java for QA
Java for QA workshops



## Prerequisites

- Java 11 (or newer)
- Intellij IDEA
- Gradle/Maven



### Java installation

On Mac using homebrew:
```bash
brew cask install java11
```

Check installed version:
```bash
java -version
```



### Build tools

Gradle:

```bash
brew install gradle
gradle --version
```

Maven:

```bash
brew install maven
mvn --version
```
